## Requirements

* HTTP Server. For example: Apache. Having mod_rewrite is preferred, but by no means required.
* PHP 5.5.9 or greater (including PHP 7).
* mbstring PHP extension
* intl PHP extension
* PDO extensions (php_pdo_mysql)
